# EZStay API

## Routes

### /api/v1/hotels/

| METHOD     | Param | Description        |
| ---------- | ----- | ------------------ |
| **GET**    | none  | Get all hotels     |
| **POST**   | none  | Create a hotel     |
| **GET**    | id    | Get hotel by id    |
| **PATCH**  | id    | Update hotel by id |
| **DELETE** | id    | DELETE a hotel     |
